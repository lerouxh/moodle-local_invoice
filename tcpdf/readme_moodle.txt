TCPDF third-party library
========================

This plugin bundles the TCPDF library to generate PDF invoices.

Project homepage
----------------
https://tcpdf.org/

Source repository
-----------------
https://github.com/tecnickcom/TCPDF

Version bundled
---------------
6.10.0 (see tcpdf/VERSION)

License
-------
LGPL v3 (see tcpdf/LICENSE.TXT)

Update process
--------------
1. Download a new TCPDF release from the official repository.
2. Replace the contents of this tcpdf/ directory with the new release.
3. Update thirdpartylibs.xml with the new version and licensing details.
4. Run the Moodle plugin prechecker and ensure no new issues are introduced.
